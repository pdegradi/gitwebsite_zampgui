ZampGUI is the ultimate portable web development environment for 64-bit Windows.

If you want to:

  • Install WordPress on your machine to experiment or keep a local copy of your online site
  • Locally install applications that rely on PHP, MySQL and Apache httpd
  • Work as a web developer looking for a robust, simple, and optimized solution for 64-bit Windows

ZampGUI is the ideal option. This suite eliminates the complexity of configuration and allows you to get started in just a few minutes.

How to use ZampGUI

ZampGUI is portable — everything is contained in a single folder that you can move or delete without installing anything on your machine.

  1. Download and unzip the file
  2. Run the executable ZampGUI3_Run.exe in the root of the folder
